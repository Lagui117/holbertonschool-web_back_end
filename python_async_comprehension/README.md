# Python - Async Comprehension

This project contains tasks related to asynchronous generators and comprehensions in Python using `asyncio`.

## Tasks

0. **Async Generator** - Write a coroutine `async_generator` that yields 10 random numbers
1. **Async Comprehensions** - Write a coroutine `async_comprehension` that collects values using async comprehension
2. **Run time for four parallel comprehensions** - Write a coroutine `measure_runtime` that measures execution time

## Requirements

- All files will be interpreted/compiled on Ubuntu 20.04 LTS using python3 (version 3.9)
- All files should end with a new line
- The first line of all files should be exactly `#!/usr/bin/env python3`
- Code should use the pycodestyle style (version 2.5.x)
- All functions and coroutines must be type-annotated
- All modules should have documentation
- All functions should have documentation
