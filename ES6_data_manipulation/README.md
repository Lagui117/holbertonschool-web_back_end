# ES6 Data Manipulation

This project contains exercises on ES6 data manipulation methods including map, filter, and reduce.

## Tasks

### 0. Basic list of objects
Create a function `getListStudents` that returns an array of student objects.

### 1. More mapping
Create a function `getListStudentIds` that returns an array of ids using the map function.

## Requirements
- All files are executed on Ubuntu 18.04 LTS using NodeJS 12.11.x
- All files use the `.js` extension
- Code is tested using Jest
- Code is verified against lint using ESLint
